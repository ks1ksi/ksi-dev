## 플로이드 - 와샬 (Floyd - Warshall)

모든 정점에서, 다른 모든 정점까지의 최단거리를 $O(N^3)$ 에 구할 수 있다.
반복문 순서에 유의한다. $dist(i, j)$와 $dist(i, k) + dist(k, j)$를 비교하는데, $k$를 고정시켜놓고 $i, j$를 움직인다.
다이나믹 프로그래밍의 일종.

```cpp
while (r--) {
	int a, b, c;
	cin >> a >> b >> c;
	dist[a-1][b-1] = c;
	dist[b-1][a-1] = c;
}

for (int i = 0; i < n; i++) {
	for (int j = 0; j < n; j++) {
		if (!dist[i][j]) dist[i][j] = 1e9;
		if (i == j) dist[i][j] = 0;
	}
}

// i -> j vs i -> k -> j
for (int k = 0; k < n; k++) {
	for (int i = 0; i < n; i++) {
		for (int j = 0; j < n; j++) {
			dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
		}
	}
}
```
